# EidolonOS (EOS)

A GitHub-based personal site builder. Clone, configure, and deploy — no build tools, no CLI, just edit a JSON file and you're live.

**[Live Example →](https://supernova0866.github.io/Lore)**

---

## Get Started

**1. Fork this repo**
Click the Fork button at the top right. Rename it to anything you want — your site will be at:
```
https://yourusername.github.io/reponame
```

**2. Enable GitHub Pages**
Go to your repo → Settings → Pages → Source → set to `main` branch → Save.

**3. Edit `config.json`**
Fill in your name, socials, enable modules. See [Config.md](Config.md) for every field explained.

**4. You're live**
Visit your GitHub Pages URL. If something looks outdated, hard refresh with `Ctrl + Shift + R`.

---

## Structure

```
your-repo/
├── index.html       ← don't touch
├── config.json      ← your config lives here
├── Config.md        ← quick config reference
└── data/
    ├── projects.json    ← your projects (if module enabled)
    ├── things.json      ← your hobbies (if module enabled)
    └── quotes.json      ← your quotes (if quotes enabled)
```

Only touch `config.json` and the files inside `data/`. Everything else is hands-off.
